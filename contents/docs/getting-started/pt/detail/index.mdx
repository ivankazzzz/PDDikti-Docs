Dokumentasi ini menjelaskan cara menggunakan endpoint `Detail PT` dan bagaimana struktur response yang akan Kamu terima. Dengan endpoint ini, Kamu dapat melihat detail informasi perguruan tinggi berdasarkan `id_pt`.

## Endpoint

Untuk mendapatkan detail data Perguruan Tinggi, Kamu dapat menggunakan endpoint berikut:

```plaintext
GET /pt/detail/<str:id_pt>
```

Pada endpoint ini, `id_pt` adalah ID unik dari perguruan tinggi yang ingin dicari detailnya.

## Contoh Endpoint

Untuk memberikan gambaran yang lebih jelas, berikut adalah contoh penggunaan endpoint:

```plaintext
GET /pt/detail/IIrmU70D6TnWs79XzUmnBLmgfQLYxkZzUeQjVW4cA9_WKIIB0oJfhVSNS865-zzStoB4tg==
```

> Dalam contoh ini, API akan mencoba mengambil detail data perguruan tinggi yang memiliki ID `IIrmU70D6TnWs79XzUmnBLmgfQLYxkZzUeQjVW4cA9_WKIIB0oJfhVSNS865-zzStoB4tg==`.

    ```python:main.py showLineNumbers {13-14}
            return response.json()
        except requests.exceptions.RequestException as e:
    ```

    ```php:main.php showLineNumbers {15-16}
    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/pt/detail";
     $id_pt = "IIrmU70D6TnWs79XzUmnBLmgfQLYxkZzUeQjVW4cA9_WKIIB0oJfhVSNS865-zzStoB4tg==";
    $api_url = $url . '/' . urlencode($id_pt);
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
    ```

    - **Mengirim Permintaan**: Fungsi `fetch_api` mengirim permintaan GET ke URL API dan mengembalikan data JSON.

    - **Mencetak Hasil**: Data yang diambil dicetak dalam format JSON yang rapi atau pesan kesalahan jika data tidak ditemukan.
