Selamat datang di Website **PDDIKTI Docs**! PDDIKTI Docs menyediakan dokumentasi API sangat lengkap seperti pada website resmi yang ada di [PDDIKTI KEMDIKTISAINTEK](https://pddikti.kemdiktisaintek.go.id/). API yang tersedia seperti fitur pencarian, detail, dan statistik dari perguruan tinggi, program studi, dosen, dan mahasiswa.

  PDDIKTI Docs ini tidak official dan tidak berkolaborasi dari PDDIKTI KEMDIKTISAINTEK.
</Note>

## Gambaran Proyek

**PDDIKTI Docs** memungkinkan pengguna untuk mempelajari dan mengambil data dari [PDDIKTI KEMDIKTISAINTEK](https://pddikti.kemdiktisaintek.go.id/) tanpa memerlukan autentikasi dan validasi dari website resmi yang ada. Sehingga PDDIKTI Docs ini dapat memberikan cara menggunakan API dengan mudah dan simpel oleh semua pihak.

### Fitur Utama

| **Fitur**                     | **Deskripsi**                                          |
|-------------------------------|----------------------------------------------------------|
| Pencarian                     | Pencarian semua, perguruan tinggi, program studi, dosen, dan mahasiswa. |
| Perguruan Tinggi (PT)         | Detail perguruan tinggi (PT), list program studi (prodi), rasio dosen dengan mahasiswa, biaya kuliah, logo, dan masih banyak lagi. |
| Program Studi (prodi)         | Detail prodi, deskripsi, jumlah mahasiwa dan dosen, riwayat, jumlah dosen, dan lain-lain. |
| Dosen                         | Detail dosen dan histori (pendidikan, mengajar, penelitian, pengabdian, karya, dan paten) |
| Mahasiswa                     | Detail mahasiswa. |
| Statistik | Mahasiswa, dosen, pergurutan tinggi, prodi. Mencakup jumlah total, jumlah aktif, jumlah berdasarkan gender, akreditasi, dan lain-lain. |

## Teknologi & Library

PDDIKTI Docs memanfaatkan teknologi mutakhir untuk memastikan kinerja dan fleksibilitas:

- **Next.js 14** - Framework React yang kuat dan dioptimalkan untuk produksi.
- **Tailwind CSS** - Styling berbasis utilitas untuk desain yang cepat dan bersih.
- **Shadcn-UI** - Komponen elegan dan aksesibel untuk tampilan yang rapi.
- **next-mdx-remote** - Mendukung MDX untuk konten Markdown yang dinamis dan interaktif.
- **Django** - Framework web tingkat tinggi yang mendorong pengembangan cepat dan desain yang bersih.
- **Django REST framework** - Toolkit yang kuat dan fleksibel untuk membangun API web.
