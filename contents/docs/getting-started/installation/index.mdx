---
title: Instalasi
description: Panduan instalasi untuk menggunakan API.
---

Menyiapkan PDDikti Docs untuk digunakan adalah sangat mudah, dengan melakukan instalasi `requests` Kamu dapat menggunakan API secara langsung. Berikut adalah opsi yang dapat digunakan:

## Google Colab
Untuk memulai, Kamu dapat mengikuti langkah-langkah berikut:
<Stepper>
  <StepperItem title="Buka Google Colab">
    Kamu dapat membuka Google Colab melalui [tautan ini](https://colab.research.google.com/).

    ```plaintext
    https://colab.research.google.com/
    ```
  </StepperItem>

  <StepperItem title="Buat proyek baru">
    Buat proyek baru dengan menekan tombol `New Notebook`.
  </StepperItem>

  <StepperItem title="Buat cell baru">
    Cell baru dapat dibuat dengan menekan `add code`.
  </StepperItem>

  <StepperItem title="Install library requests">
    Lakukan instalasi `requests` dengan mengetikkan ini di cell tersebut:

    ```plaintext
    !pip install requests
    ```
    <Note type="danger" title="Ingat">
      Pastikan Kamu menjalankan cell tersebut agar dapat melakukan instalasi.
    </Note>

    <Note type="note" title="Perhatian">
      Khusus pada endpoint `logo`, kamu harus menginstall library `pillow` dengan cara `!pip install pillow`.
    </Note>

  </StepperItem>

  <StepperItem title="Hore!">
    <Note type="success" title="Berhasil">
      Hore! Kamu berhasil melakukan instalasi dengan benar dan dapat digunakan!
    </Note>
  </StepperItem>
</Stepper>

## Visual Studio Code
Untuk memulai, Kamu dapat mengikuti langkah-langkah berikut:
<Stepper>
  <StepperItem title="Buka Visual Studio Code">
    <Note type="warning" title="Peringatan">
      Jika Kamu belum memiliki Visual Studio Code, Kamu dapat menginstall terlebih dahulu melalui tautan [berikut](https://code.visualstudio.com/download/) atau menggunakan Google Colab.
    </Note>
    Buka Visual Studio Code seperti meembuka aplikasi lain yang ada di komputer/pc Kamu.
  </StepperItem>

  <StepperItem title="Buat virtual environment">
    Buka terminal pada Visual Studio Code dan ketikkan text berikut:
    ```plaintext
    virtualenv venv
    ```
  </StepperItem>

  <StepperItem title="Aktifkan virtual environment">
    Untuk windows:
    ```plaintext
    venv\scripts\activate
    ```

    Untuk mac/linux:
    ```plaintext
    source venv/bin/activate
    ```
  </StepperItem>

  <StepperItem title="Install library requests">
    Lakukan instalasi `requests` dengan mengetikkan ini di terminal:

    ```plaintext
    pip install requests
    ```
    <Note type="note" title="Perhatian">
      Khusus pada endpoint `logo`, kamu harus menginstall library `pillow` dengan cara `!pip install pillow`.
    </Note>

    <Note type="warning" title="Ingat">
      Kamu dapat mengecek apakah library telah terinstall di virtual environment dengan menjalankan ini di terminal `pip freeze`.
    </Note>

  </StepperItem>

  <StepperItem title="Hore!">
    <Note type="success" title="Berhasil">
      Hore! Kamu berhasil melakukan instalasi dengan benar dan dapat digunakan!
    </Note>
  </StepperItem>
</Stepper>

Dengan setup instalasi ini, Kamu telah siap untuk menggunakan PDDikti Docs dan mencoba API yang Kamu ingin coba.
