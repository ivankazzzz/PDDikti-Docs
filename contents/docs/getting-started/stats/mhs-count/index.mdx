Dokumentasi ini menjelaskan cara menggunakan endpoint `Statistik Jumlah Mahasiswa` dan bagaimana struktur response yang akan Anda terima. Dengan endpoint ini, Anda dapat melihat informasi jumlah total mahasiswa terdaftar.

## Endpoint

Untuk mendapatkan informasi jumlah total mahasiswa terdaftar, Anda dapat menggunakan endpoint berikut:

```plaintext
GET /stats/mhs-count/
```

Pada endpoint ini, tidak ada parameter yang diperlukan.

## Contoh Endpoint

Untuk memberikan gambaran yang lebih jelas, berikut adalah contoh penggunaan endpoint:

```plaintext
GET /stats/mhs-count/
```

> Dalam contoh ini, API akan mencoba mengambil informasi jumlah total mahasiswa terdaftar.

    ```python:main.py showLineNumbers {13-14}
    import requests
    import json

    def fetch_api(api_url):
        try:
            response = requests.get(api_url)
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            print(f"Error: {e}")
            return None

    url = "https://api-pddikti.vercel.app/stats/mhs-count"
    api_url = f"{url}/"
    data = fetch_api(api_url)

    print(json.dumps(data, indent=4) if data else "No data fetched.")
    ```

    - **Mengirim Permintaan**: Fungsi `fetch_api` mengirim permintaan GET ke URL API dan mengembalikan data JSON.
    - **Menggabungkan URL**: `url` digabungkan untuk membentuk URL lengkap.

    Tidak ada parameter yang dibutuhkan.

    ```php:main.php showLineNumbers {15-16}
    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/stats/mhs-count";
    $api_url = $url . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
    ```

    - **Mengirim Permintaan**: Fungsi `fetch_api` mengirim permintaan GET ke URL API dan mengembalikan data JSON.
    - **Menggabungkan URL**: `url` digabungkan untuk membentuk URL lengkap.
    - **Mencetak Hasil**: Data yang diambil dicetak dalam format JSON yang rapi atau pesan kesalahan jika data tidak ditemukan.
