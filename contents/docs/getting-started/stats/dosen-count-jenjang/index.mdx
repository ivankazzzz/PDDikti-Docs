Dokumentasi ini menjelaskan cara menggunakan endpoint `Statistik Jumlah Dosen Berdasarkan Jenjang Pendidikan` dan bagaimana struktur response yang akan Kamu terima. Dengan endpoint ini, Kamu dapat melihat informasi jumlah total dosen terdaftar berdasarkan jenjang pendidikan.

## Endpoint

Untuk mendapatkan informasi jumlah total dosen terdaftar berdasarkan jenjang pendidikan, Kamu dapat menggunakan endpoint berikut:

```plaintext
GET /stats/dosen-count-jenjang/
```

Pada endpoint ini, tidak ada parameter yang diperlukan.

    ```php:main.php showLineNumbers {15}
    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/stats/dosen-count-jenjang";
    $api_url = $url . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
    ```

    -   **Mengirim Permintaan**: Fungsi `fetch_api` mengirim permintaan GET ke URL API dan mengembalikan data JSON.
    *   **Menggabungkan URL**: `url` digabungkan untuk membentuk URL lengkap.
    -   **Mencetak Hasil**: Data yang diambil dicetak dalam format JSON yang rapi atau pesan kesalahan jika data tidak ditemukan.
