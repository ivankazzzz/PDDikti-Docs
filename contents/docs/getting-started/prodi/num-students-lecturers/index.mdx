Dokumentasi ini menjelaskan cara menggunakan endpoint `Jumlah Mahasiswa dan Dosen` dan bagaimana struktur response yang akan Anda terima. Dengan endpoint ini, Anda dapat melihat informasi jumlah mahasiswa dan dosen per semester pada suatu program studi berdasarkan `id_prodi`.

## Endpoint

Untuk mendapatkan informasi jumlah mahasiswa dan dosen per semester pada suatu Program Studi, Anda dapat menggunakan endpoint berikut:

```plaintext
GET /prodi/num-students-lecturers/<str:id_prodi>/
```

Pada endpoint ini, `id_prodi` adalah ID unik dari program studi yang ingin dicari informasi jumlah mahasiswa dan dosennya.

## Contoh Endpoint

Untuk memberikan gambaran yang lebih jelas, berikut adalah contoh penggunaan endpoint:

    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/prodi/num-students-lecturers";
     $id_prodi = "sIU3_7yvFq6ot-F8Hu3d1h0zWf21NkeT4wd-KWCCCFLo26jwAqZAMjDd_xDuloWA8CzNkg==";
    $api_url = $url . '/' . urlencode($id_prodi) . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
