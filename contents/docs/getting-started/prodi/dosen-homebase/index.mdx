Dokumentasi ini menjelaskan cara menggunakan endpoint `Dosen Homebase Prodi` dan bagaimana struktur response yang akan Anda terima. Dengan endpoint ini, Anda dapat melihat informasi dosen homebase pada suatu program studi berdasarkan `id_prodi` dan `id_thsmt`.

## Endpoint

Untuk mendapatkan informasi dosen homebase pada suatu Program Studi, Anda dapat menggunakan endpoint berikut:

```plaintext
GET /prodi/dosen-homebase/<str:id_prodi>/<str:id_thsmt>/
```

Pada endpoint ini, `id_prodi` adalah ID unik dari program studi dan `id_thsmt` adalah ID unik dari tahun semester yang ingin dicari informasi dosen homebasenya.

    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/prodi/dosen-homebase";
    $id_prodi = "sIU3_7yvFq6ot-F8Hu3d1h0zWf21NkeT4wd-KWCCCFLo26jwAqZAMjDd_xDuloWA8CzNkg==";
     $id_thsmt = "20241";
    $api_url = $url . '/' . urlencode($id_prodi) . '/' . urlencode($id_thsmt) . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
