<Note type="danger" title="Perhatian">
 API akan menampilkan website jika dibuka langsung. Namun, saat menggunakan Postman atau alat lainnya, data akan otomatis dalam format JSON. Untuk memastikan, tambahkan `?format=json`.
</Note>

<Note type="success" title="Contoh">
 `https://api-pddikti.ridwaanhall.com/search/pt/uty/?format=json`.
</Note>
