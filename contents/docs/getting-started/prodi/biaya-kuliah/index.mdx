Dokumentasi ini menjelaskan cara menggunakan endpoint `Biaya Kuliah Prodi` dan bagaimana struktur response yang akan Kamu terima. Dengan endpoint ini, Kamu dapat melihat informasi terkait range biaya kuliah pada suatu program studi berdasarkan `id_prodi`.

## Endpoint

Untuk mendapatkan informasi range biaya kuliah pada suatu Program Studi, Kamu dapat menggunakan endpoint berikut:

```plaintext
GET /prodi/biaya-kuliah/<str:id_prodi>/
```

Pada endpoint ini, `id_prodi` adalah ID unik dari program studi yang ingin dicari informasi biaya kuliahnya.

## Contoh Endpoint

Untuk memberikan gambaran yang lebih jelas, berikut adalah contoh penggunaan endpoint:


> Dalam contoh ini, API akan mencoba mengambil informasi range biaya kuliah pada program studi yang memiliki ID `sIU3_7yvFq6ot-F8Hu3d1h0zWf21NkeT4wd-KWCCCFLo26jwAqZAMjDd_xDuloWA8CzNkg==`.

    ```php:main.php showLineNumbers {15-16}
    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/prodi/biaya-kuliah";
     $id_prodi = "sIU3_7yvFq6ot-F8Hu3d1h0zWf21NkeT4wd-KWCCCFLo26jwAqZAMjDd_xDuloWA8CzNkg==";
    $api_url = $url . '/' . urlencode($id_prodi) . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
    ```
