Dokumentasi ini menjelaskan cara menggunakan endpoint `Paten dan Hak Cipta Dosen` dan bagaimana struktur response yang akan Kamu terima. Dengan endpoint ini, Kamu dapat melihat informasi paten dan hak cipta dosen berdasarkan `id_dosen`.

## Endpoint

Untuk mendapatkan informasi paten dan hak cipta Dosen, Kamu dapat menggunakan endpoint berikut:

```plaintext
GET /dosen/paten/<str:id_dosen>/
```

Pada endpoint ini, `id_dosen` adalah ID unik dari dosen yang ingin dicari informasi paten dan hak ciptanya.

## Contoh Endpoint

Untuk memberikan gambaran yang lebih jelas, berikut adalah contoh penggunaan endpoint:

```plaintext
GET /dosen/paten/AtnqXm_IHYxyGVe_q-YfgBBVmNdZ6DQxS1WpDkwFJxrbj8elL8B0y3d9VoLwmXHx8hzD8w==/
```

> Dalam contoh ini, API akan mencoba mengambil informasi paten dan hak cipta dosen yang memiliki ID `AtnqXm_IHYxyGVe_q-YfgBBVmNdZ6DQxS1WpDkwFJxrbj8elL8B0y3d9VoLwmXHx8hzD8w==`.

    ```php:main.php showLineNumbers {15-16}
    <?php
    function fetch_api($api_url) {
        $ch = curl_init();
        curl_setopt($ch, CURLOPT_URL, $api_url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
        $response = curl_exec($ch);
        if (curl_errno($ch)) {
            echo 'Error: ' . curl_error($ch);
            return null;
        }
        curl_close($ch);
        return json_decode($response, true);
    }

    $url = "https://api-pddikti.vercel.app/dosen/paten";
    $id_dosen = "AtnqXm_IHYxyGVe_q-YfgBBVmNdZ6DQxS1WpDkwFJxrbj8elL8B0y3d9VoLwmXHx8hzD8w==";
    $api_url = $url . '/' . urlencode($id_dosen) . '/';
    $data = fetch_api($api_url);

    if ($data) {
        echo '<pre>' . json_encode($data, JSON_PRETTY_PRINT) . '</pre>';
    } else {
        echo "No data fetched.";
    }
    ?>
    ```

    - **Mengirim Permintaan**: Fungsi `fetch_api` mengirim permintaan GET ke URL API dan mengembalikan data JSON.
    - **Menggabungkan URL**: `url` dan `id_dosen` digabungkan untuk membentuk URL lengkap.
    - **Mencetak Hasil**: Data yang diambil dicetak dalam format JSON yang rapi atau pesan kesalahan jika data tidak ditemukan.
